Virtual Environment

To create virtual environment on Linux and Windows

---------------------- Linux --------------------------------
- Check Python version
    python3 --version
- Update your system packages
    sudo apt-get update
- Install python3 environment packages
    sudo apt install python3 python3-venv
- Create virtual environment
    python3 -m venv myenvironment
- Activate python3 virtual environment
    source myenvironment/bin/activate

Install the above Listed Packages in the virtual environment

    Pandas
    Seaborn
    Sklearn
